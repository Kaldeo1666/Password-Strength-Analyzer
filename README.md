# Password Strength Analyzer 🔐

A simple web-based Password Strength Analyzer built using HTML, CSS, and JavaScript.

## 🚀 Features
- Checks password length
- Detects uppercase letters
- Detects numbers
- Detects special characters
- Displays strength level (Weak / Medium / Strong)
- Dynamic strength bar indicator

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
